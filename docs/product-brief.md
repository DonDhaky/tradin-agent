# Product Brief

## Product Name

Tradein Agent V1

## Goal

Provide a fast, trustworthy way for users to submit second-hand tech products for trade-in review.

## Primary User

A customer who wants to sell or trade in a used tech device and needs a simple intake flow.

## V1 Outcomes

- Present a premium landing page
- Let users create an account or sign in
- Restrict the dashboard to authenticated users
- Collect key product details through a structured form
- Store each submission in Firestore

## V1 Non-Goals

- Automated pricing or AI estimation
- Marketplace comparisons
- Device scraping or enrichment
- Messaging or spreadsheet integrations
- Internal admin tooling

## Core User Journey

1. User lands on the homepage and understands the value proposition.
2. User signs up or logs in.
3. User reaches the protected dashboard.
4. User submits a device through the intake form.
5. Submission is stored and visible in the dashboard history.

## Product Principles

- Keep the experience calm and premium.
- Reduce form friction while collecting enough detail for manual follow-up.
- Keep the repository ready for future AI augmentation without implementing it now.
